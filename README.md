# 8088-OS
A operating system for the i8088

This is targeting the 8088 CPU in vintage PCs

It might work on:

- 8086
- 80186
- 80188
- 80286

You'll need:

- QEMU qemu-system-i386 or qemu-system-i8086 or maybe qemu-system-x86

also, 

- NASM
- dd
- gdb

this OS is completely written in assembly. 

Assemble this with nasm -f to a .bin

dd to a .img
